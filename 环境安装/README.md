# Langchain 环境搭建

## 下载


```shell
# 下载 langchain
pip install langchain

# 查看当前环境下的包
pip list

# 快照
pip freeze > pypackage-langch.txt

# 安装 commmunity
pip install langchain-community

# 快照
pip freeze > pypackage-langch-community.txt

# 安装 langchain-ollama
pip install langchain-ollama

# 安装 deepseek api 接口
pip install langchain-deepseek

# 安装向量数据库 chromadb
pip install chromadb

# 安装 langchain-chroma 用于langchain 与 chromadb 的连接
pip install langchain-chroma
```
